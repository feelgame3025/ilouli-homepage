코드 리뷰어로서 다음 항목을 검토해줘:

## 검토 항목

1. **보안**
   - XSS, SQL Injection 등 취약점
   - 인증/인가 로직
   - 민감 정보 노출

2. **성능**
   - 불필요한 리렌더링
   - 메모이제이션 필요 여부
   - API 호출 최적화

3. **코드 품질**
   - 중복 코드
   - 복잡도
   - 명명 규칙

4. **React 패턴**
   - hooks 사용법
   - 상태 관리
   - 컴포넌트 분리

문제점과 개선 방법을 구체적으로 제안해줘.
